HTML Highlighting Made Easy

jquery-highlighter is an open source jquery plugin which lets you save and draw highlights on web pages using spans. Works across HTML elements like this text in bold tags and this text in italics

Why does this exist?

Dealing with highlighting in browsers is diffucult. If you use the built-in browser highlight to mark parts of the page, you can only have 1 active highlight marked, and if the user clicks anywhere else on the page the highlight disappears. In addition, there's no way to add mouseover or other listeners to built-in browser. Furthermore, the interface for dealing with and manipulating selections is difficult to use.

Saving and drawing highlights using HTML is definitely the way to go, but this, too, presents difficulties. HTML is a fundamentally a tree structure where each element has exactly 1 parent. However, selections break this structure because the user can select text which spans multiple HTML elements, and may even end in the middle of an element.

What does jQuery-Highlighter do?

jQuery-highlighter handles all the complexity of marking and drawing selections using HTML spans rather than the built-in highlighter. In addition, it simplifies the interface to read in selection bounds by returning a start index and end index of a selection relative to a parent HTML element as if there were only flat text inside, regardless of the complexity of the actual underlying HTML.
